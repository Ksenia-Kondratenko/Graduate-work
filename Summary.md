# Отчёт о проведённой автоматизации

В данном проекте планировалось разработать и выполнить тест-кейсы, охватывающие позитивные и негативные проверки формы оплаты тура, а также проверку взаимодействия с базой данных. По результатам проведена большая часть запланированных тестов, за исключением проверки взаимодействия с базой данных.


Основной причиной невыполнения проверки взаимодействия с базой данных стали технические трудности, связанные с неправильной настройкой конфигурации окружения. 
В частности:
- ошибки настройки конфигурации контейнеров Docker, а именно: ошибка "Cannot start Docker Compose application. Reason: compose [start] exit status 1. Container mysql_db Starting Error response from daemon: ports are not available: exposing port TCP 0.0.0.0:3306 -> 127.0.0.1:0: listen tcp 0.0.0.0:3306: bind: Only one usage of each socket address (protocol/network address/port) is normally permitted".
Сервер MySQL не смог стартовать, так как порт 3306 уже занят другим приложением.
- ошибки настройки баз данных: ошибка "2003, "Can't connect to MySQL server on 'mysql_db' ([Errno 11001] getaddrinfo failed)"" - приложение не может найти сервер MySQL по имени хоста mysql_db. Ошибка "1146, "Table 'app.payment_entity' doesn't exist" - отсутствие таблицы в базе данных.


Сработавшие риски:
- неправильная конфигурация окружения: ошибки в настройке Docker, влияющие на работу контейнеров и взаимодействие между ними.
- технические сложности: проблемы с настройкой базы данных и доступом к ней из тестов.


Общий итог по времени.
Присутствует расхождение по времени в большую сторону в рамках этапов развёртывания и написания тестов. Основными причинами превышения планируемого времени стали:
- развертывание и настройка Docker, настройка взаимодействия с базой данных.
- сложности с написанием тестов из-за объёма и специфики проекта.
