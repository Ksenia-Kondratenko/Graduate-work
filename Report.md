# Отчёт о проведённом тестировании

1. Краткое описание.

В рамках данного проекта проведено автоматизированное тестирование приложения оплаты тура с использованием Selenium. Были проверены основные позитивные и негативные сценарии использования приложения при оплате тура без кредита, оплате тура с использованием кредита, а также взаимодействие с базой данных.

2. Количество тест-кейсов: 69


3. Процент успешных и неуспешных тест-кейсов:
- Успешных: 75,36% (52)
- Неуспешных: 18,84% (13)
- С ошибкой: 5,80% (4)

4. Общие рекомендации:
- отладка неустойчивых тестов: скорректировать тесты, завершающиеся с ошибкой.
